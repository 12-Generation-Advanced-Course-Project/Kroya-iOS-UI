## Getting Started

To get a local copy up and running follow these simple steps.


### Installation

1. Clone the repo
   
```sh
   git clone https://github.com/12-Generation-Advanced-Course-Project/Kroya-iOS-UI.git
```
2. install pod

```sh
  pod install
```
### Start

1. Start the project
   Open Xcode > Choose folder of the cloned project
